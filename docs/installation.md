# Installation Guide

This guide outlines the protocol required to integrate **SolInPy** into your development environment.

## Prerequisites

Ensure your system meets the following enterprise requirements before installing the framework:

* **Python**: Version 3.8 or higher.
* **Environment**: A virtual environment (`venv`, `pipenv`, or `poetry`) is strongly recommended to isolate project dependencies.

## Standard Installation

The most efficient method to install SolInPy is via the Python Package Index (PyPI). Execute the following command within your active terminal:

```bash
pip install solinpy

Installation from Source
